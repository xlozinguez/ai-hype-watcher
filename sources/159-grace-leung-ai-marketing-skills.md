# 159: Claude Skills — Build Your First AI Marketing Team

> **Creator**: Grace Leung | **Platform**: YouTube | **Date**: 2026-02-21 | **Duration**: 16:42

## Summary

Grace Leung demonstrates how to build a multi-skill marketing workflow using Claude Code skills, creating five specialized skills that cover research/strategy, content creation, creative design, data analysis, and campaign presentation. The video shows skills working individually, in combination, and orchestrated through sub-agents for complex multi-step marketing deliverables.

The key insight is that skills encode domain expertise — SOPs, frameworks, brand standards — making them portable across projects and brands. Grace demonstrates packaging skills into a plugin for distribution across different brand projects and even different tools like Claude Desktop, showing the composability and reusability of skill-based workflows.

## Key Concepts

### Skills as Encoded Expertise

Skills are not just automation scripts but encoded marketing knowledge — SOPs, proven frameworks, storytelling patterns, and brand standards. By creating skills from existing best-practice documents and example content, marketers capture institutional knowledge in a portable, reusable format. The skill creator tool analyzes reference materials and builds structured workflows that any agent can follow.

### Multi-Skill Orchestration

When given a complex task requiring multiple skills, Claude acts as a team lead — reading context, selecting relevant skills, and orchestrating execution. For example, a campaign launch task triggers both the social media content skill and the creative designer skill without the user having to specify the workflow. The agent determines the most efficient way to combine skills based on the task requirements.

### Skill Portability via Plugins

Skills can be packaged into plugins — bundled packages that include all skill files and reference materials. This enables distributing a complete marketing workflow to new brand projects or even different tools. The same skill set produces on-brand outputs for different clients by reading each project's brand context files, demonstrating true reusability.

## Practical Takeaways

- **Build skills from existing SOPs**: Convert your team's best practices and standard procedures into Claude skills for consistent, repeatable execution
- **Show, don't just tell**: Include example outputs and reference materials when creating skills — pattern matching from examples produces better results than instructions alone
- **Use sub-agents for parallel work**: For complex deliverables (strategy + data analysis + presentation), let Claude spin up parallel sub-agents each equipped with relevant skills
- **Package skills as plugins**: Bundle related skills into plugins for easy distribution across projects, brands, or team members
