A rapid-fire AI news roundup covering a leaked Anthropic model tier called "Claude Mythos" (reportedly a 10 trillion parameter model), positioned above the existing Opus flagship, alongside a new intermediate tier called "Capabara." Early testers and Fortune reporting suggest capabilities that are qualitatively discontinuous from current models, prompting Anthropic to plan a slow rollout due to misuse and security concerns. On the tooling side, several meaningful updates shipped: OpenAI added a plugin/workflow ecosystem to Codex, directly competing with Claude Code; GLM 5.1 from Zhipu launched as a competitive open-source agentic model approaching proprietary benchmarks; and Google DeepMind released Gemini 3.1 Flash Live for real-time voice/vision agents. Claude Code itself received several updates including cloud-based PR autofix, temporary session limits during peak hours, and an auto mode that eliminates constant permission prompts.

Smaller but notable items include: ARC-AGI 3 raising the benchmark ceiling with sub-1% AI solve rates, Anthropic's "Operon" scientific research agent for Claude Desktop, Cursor's Composer 2 revealed to be a fine-tuned Kimi K2.5 model (not an internally-built frontier model as implied), Mistral's open-weight TTS model, ElevenLabs' agent-first CLI, and the impending shutdown of the Sora app.

## Key Concepts

### Claude Mythos / Capabara Leaked Model Tiers
A purported internal leak revealed two new Anthropic model tiers. "Mythos" sits above all existing models including Opus, while "Capabara" forms a new class between the two. Reported enhancements include coding, academic reasoning, and notably large cybersecurity capability gains. Anthropic is allegedly planning a cautious, staged rollout due to safety and misuse risk — a pattern consistent with their published responsible scaling policies. The framing from the video acknowledges this may also be a deliberate marketing play to build anticipation.

### ARC-AGI 3 as a Meaningful Intelligence Benchmark
The third iteration of the ARC-AGI benchmark is designed explicitly to resist overfitting and memorization — the core failure mode of previous benchmarks. Current frontier models score under 1%, while humans achieve 100% first-attempt solve rates. The benchmark tests agentic reasoning in interactive environments with no prior training on the tasks. Future iterations are planned to incorporate commercial video games as evaluation environments, pushing toward testing adaptive, embodied-style reasoning.

### Codex Plugin Ecosystem vs. Claude Code
OpenAI transformed Codex from a prompt-response tool into a full execution platform with a plugin and workflow gallery. Users can launch pre-built, one-click runnable workflows for tasks like iOS app building, data analysis, and report generation. This is framed as a direct competitive move against Claude Code's agentic coding position, shifting Codex from a single-tool interaction to a development platform with composable, extensible workflows.

### Claude Code Auto Mode and Session Limits
Two operational updates to Claude Code: (1) Auto mode introduces a built-in classifier that categorizes every action as safe (runs instantly) or risky (blocked), eliminating constant permission interruptions while preserving control — a meaningful UX improvement for autonomous workflows. (2) A temporary tightening of 5-hour session limits during peak weekday hours for free, pro, and max subscribers, with weekly totals unchanged, reflecting infrastructure demand pressure.

### Cursor Composer 2 / Kimi K2.5 Disclosure Gap
Cursor released "Composer 2" as a claimed frontier-level proprietary coding model benchmarked on internal evals (Cursor Bench). Community investigation revealed it was actually a fine-tuned version of the open-source Kimi K2.5 model. The incident highlights a recurring tension in the AI tooling space: companies releasing fine-tuned open-source models under proprietary branding without disclosure, creating misleading capability narratives.

## Practical Takeaways

- **Watch Anthropic's staged rollout cadence**: If Mythos/Capabara rumors are accurate, expect intermediate releases (Sonnet 5 or Opus 5) before the new tier lands. Plan capability evaluations accordingly rather than waiting for the ceiling model.
- **Claude Code auto mode is production-relevant now**: The removal of constant permission prompts via classifier-based action gating makes Claude Code meaningfully more viable for unattended/autonomous workflows — worth re-evaluating if friction was previously a blocker.
- **ARC-AGI 3 is a credible progress signal**: Unlike saturated benchmarks, sub-1% AI performance on ARC-AGI 3 means it will provide genuine signal as models improve. Worth tracking as a leading indicator of genuine reasoning capability gains, not just benchmark gaming.
- **Scrutinize model provenance claims**: The Cursor/Kimi K2.5 case is a reminder to verify whether "proprietary" AI coding tools are fine-tuned open-source models. This affects decisions about fine-tuning, self-hosting, and vendor lock-in.
- **Codex plugin ecosystem deserves evaluation**: For teams currently using Claude Code, the Codex workflow gallery represents a legitimately different UX paradigm (pre-built executable workflows vs. freeform agentic prompting). Worth hands-on comparison depending on use case structure.
